# react-native-easy-storage

A minimalist, fast, and straightforward storage wrapper for React Native, built on top of `@react-native-async-storage/async-storage`.

## Features
- ⚡ **Type-safe:** Automatically handles JSON parsing and stringification.
- 🪶 **Lightweight:** Zero external dependencies (except peer dependencies).
- 🧼 **Clean API:** No more boilerplate code for simple local storage actions.

## Installation

```bash
npm install react-native-easy-storage
